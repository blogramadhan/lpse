INSTRUKSI PENAMBAHAN LOGO KALIMANTAN BARAT
==========================================

Untuk menampilkan logo Provinsi Kalimantan Barat (Akcaya) di aplikasi ini:

1. Siapkan file logo dengan format PNG (disarankan untuk transparansi background)

2. Simpan file logo dengan nama: logo-kalbar.png

3. Letakkan file tersebut di folder: /public/logo-kalbar.png

4. Spesifikasi logo yang disarankan:
   - Format: PNG (dengan background transparan)
   - Ukuran: Minimal 300x300 pixels
   - Resolusi: 72 DPI atau lebih tinggi untuk tampilan web

5. Setelah file logo ditempatkan, aplikasi akan otomatis menampilkan:
   - Logo di header halaman (ukuran responsif)
   - Logo sebagai favicon di browser tab
